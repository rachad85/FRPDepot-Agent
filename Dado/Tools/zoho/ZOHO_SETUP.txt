FRP DEPOT ZOHO RESTRICTED ACCESS SETUP
======================================

IMPORTANT
---------
- Use only the FRP Depot Zoho account.
- Never paste the Client Secret or Grant Code into Telegram.
- Zoho Books named writes: create customers and create draft estimates.
- Zoho Inventory named writes: create items and update item name/SKU.
- No delete, stock adjustment, order, invoice, payment, transfer, or send scope.
- Every write requires Rachad's approval.
- Zoho's items.UPDATE scope is broad, but the local named tool enforces name/SKU-only updates.

Expand the existing Self Client authorization
---------------------------------------------
1. Double-click C:\FRPDepot\PREPARE_DADO_ZOHO_ACCESS.bat

2. Sign in to the Zoho API Console using the FRP Depot Zoho account.

3. Open the existing Self Client used for Dado.

4. Click Generate Code.

5. Paste into the Scope box. The complete restricted scope list is already on the Windows clipboard.

6. Confirm there is no DELETE, .ALL, fullaccess, inventory adjustment write, order write, invoice write, or send scope.

7. Choose a 10-minute duration if Zoho offers it.

8. Add this description:
FRP Depot complete read reporting plus approved customer, draft quote, and item catalog changes

9. Click Create/Generate.

10. Keep the one-time Grant Code on screen. Do not send it in Telegram.

Connect locally
---------------
11. Double-click C:\FRPDepot\REAUTHORIZE_DADO_ZOHO.bat

12. Enter only the one-time Grant Code into the local window. It remains hidden.

13. Double-click C:\FRPDepot\CHECK_DADO_ZOHO.bat

14. Tell Dado whether the check passed. Do not send any code or credential.

The encrypted vault is stored at:
%LOCALAPPDATA%\FRPDepot-Zoho\zoho_access.dpapi
